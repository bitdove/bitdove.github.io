【English Learning】The System of English Grammar

2025/09/12

Preface

Any language has three main components: vocabulary, pronunciation, and grammar. In this article, we focus on the grammar of English.

英语的核心是动词。

简单句

长句子由简单句组合而来，拆的不能再拆的句子称为简单句。简单句表达的是“什么+怎么样”。而“什么”和“怎么样”分别对应主语和谓语这两种句子成分。即简单句=主语+谓语。主语是人或物，而谓语是广义的动作。

动作（动词）分类：

  1. 不及物动词：可以独立完成的动作；
  2. 单及物动词：有1个动作承受者；
  3. 双及物动词：有2个动作承受者；
  4. 复杂及物动词：只有1个动作承受者（但需补充）；
  5. 系动词：非“动作”，仅赋予主语某种性质或状态。
动词类型 解释 代表结构 例词 例句 解释
不及物动词 可以独立完成的动作 主语+不及物动词 sleep Mason sleeps. /
单及物动词 有1个动作承受者 主语+单及物动词+宾语 like Mason likes you. you为宾语
双及物动词 有2个动作承受者 主语+双及物动词+间接宾语+直接宾语 teach Mason teaches you English. you为间接宾语，English为直接宾语
复杂及物动词 只有1个动作承受者（但需补充说明） 主语+复杂及物动词+宾语+宾语补语 consider Mason considers you smart. you为宾语，smart为宾语补语
系动词 非“动作”，赋予主语某种性质 主语+系动词+主语补语/表语 am/is/are, look Mason is handsome. handsome为表语

谓语 ≠ 谓语动词 句子除去主语，剩下的部分都是谓语，谓语动词只是谓语的一部分。

句子成分

主语 谓语动词 宾语 宾语补语 主语补语/表语 定语 状语 同位语

复合句和复杂句

简单句互相组合可成为复合句 Compound Sentence和复杂句 Complex Sentence。 复合句是并列关系。复杂句是主从关系。

复杂句中的从句就是把简单句修改一下作为主句的句子成分。

  1. 名词性从句： 主语从句 宾语从句 表语从句 同位语从句
  2. 定语从句
  3. 状语从句

词性（十大词类）

动词 名词 冠词 代词 形容词 数词 副词 介词 叹词 连词

除了谓语动词是动词，其他句子成分都有可能包含不同的词类。

Verb Tenses

谓语动词有三大本领：

  1. 表示动作时间
  2. 表示动作状态
  3. 表示动词语气，如动作的假设、情感等。

1和2合称时态（Verb Tenses）。

助动词：没有实义，仅帮助谓语动词完成各种本领。如时态、虚拟语气、可能性、否定等等。

常见助动词：have, be, can, might, must, do。

注意，助动词也有作实义词的时候，比如have可以是“拥有”，can可以是“易拉罐”。

Simple 一般 Continuous 进行 Perfect 完成 Perfect continuous 完成进行
Past 过去 did was/were doing had done had been doing
Present 现在 do/does am/is/are doing have/has done have/has been doing
Future 将来 will do will be doing will have done will have been doing
Past future 过去将来 would do would be doing would have done would have been doing

Tips：The past future tense is not a standalone tense but rather a way to express future events relative to a point in the past.

非谓语动词

动词除了做谓语动词外，可以不同的形式作为其他句子成分（如主语、宾语、宾语补语、定语等等），这时候动词叫做非谓语动词。

有以下形式：

  • 动词不定式
  • 现在分词
  • 动名词
  • 过去分词
